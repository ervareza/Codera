# Codera - Fleet Management Platform

Codera is a modern, high-performance fleet management landing page built to showcase telematics, GPS tracking, predictive maintenance, and workforce management solutions.

## Features
- **Dynamic Animations**: Powered by `framer-motion` for a premium, engaging user experience.
- **Responsive Design**: Built with Tailwind CSS ensuring perfect rendering across all devices.
- **Modern UI/UX**: Clean aesthetics, glassmorphism hints, and optimized typography for professional branding.
- **SEO Optimized**: Follows best practices for fast loading and clear semantic structure.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Typography**: Optimized fonts via Next.js

## Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ervareza/Codera.git
   cd Codera
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result.

## Deployment

This Next.js application can be easily deployed to Vercel, the platform from the creators of Next.js.
```bash
npx vercel
```
For more detailed instructions, refer to the [Next.js deployment documentation](https://nextjs.org/docs/deployment).
